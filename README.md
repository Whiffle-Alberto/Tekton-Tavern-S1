# Tekton-Tavern-S1-modpack
modpack for my private server

Installation instructions: \n
• Download the latest release. \n
• Extract the zip file. \n
• Take all of the jar files out of the 'mods' folder and put it into your minecraft mods folder. \n
• Delete the downloaded zip file and empty mods folder. \n
